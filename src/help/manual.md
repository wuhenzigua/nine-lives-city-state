# 九命城邦 功能说明

## 核心目标
- 稳住猫国生存，扩张节点网络，推进文明时代并完成猫教飞升。

## 核心循环
- 白天：降温与准备
- 夜晚：扩张与冒险
- 黎明：统一结算与反制

## 资源
- 残羹：生存底线
- 气味：扩张/维护
- 信任：稳定与招募
- 情报：降低失败率
- 传说：轮内与局外成长货币

## 关键系统
- 节点网络：连通收益更稳，游离会带来惩罚
- 局部热点：高活跃节点更容易被针对
- 三前线：人类 / 狗群 / rival 猫群
- 命途：永久觉醒、可叠加
- 局外成长：升级、地图层级、时代项目

## 主线
- 求生时代 -> 科技时代 -> 神学时代 -> 飞升时代

## 操作建议
- 先稳残羹与气味，再扩张
- 注意热点和脆弱度，避免边界崩盘
- 传说优先用于能改善当前瓶颈的局外升级
