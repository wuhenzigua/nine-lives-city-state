# 九命城邦

基于设计文档实现的 Web MVP 原型。玩家在现代城市夹缝中经营一群流浪猫，通过 `白天潜伏 / 夜晚扩张 / 黎明结算 / 换命继承` 的循环，体验资源增长、注意度控制与气味网络维护之间的取舍。

## 已实现内容

- 6 个城市节点与可视化连线地图
- 昼夜双相循环与 120 秒阶段计时
- 5 类资源：残羹、气味、信任、情报、传说
- 4 类职业：觅食猫、外交猫、斥候猫、巡界猫
- 节点占领、失败率、注意度阈值与黎明巡查
- 气味网络连通判定、游离态与维护消耗
- 建筑系统：隐窝、气味标记、观测点、月台
- 首轮换命与两条本能路线：亲人本能、夜袭本能
- 行动日志、结算预估、最近一次黎明报告

## 交互说明

- 白天优先分配外交猫和斥候猫，压低注意度并积累信任/情报。
- 夜晚点击地图节点进行扩张，优先保证主巢到外围的连通。
- 每次黎明会消耗残羹与气味；如果注意度过高，可能触发巡查并丢失节点。
- 控制至少 4 个连通节点，在地铁废口建成月台，并完成一次稳定结算后，就能换命。

## 项目结构

- `src/game/logic.ts`：昼夜 tick、扩张、结算、换命规则
- `src/game/types.ts`：核心类型定义
- `九命城邦-设计文档.md`：原始设计文档
